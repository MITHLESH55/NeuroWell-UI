#  NeuroWell - AI Mental Wellness Assistant

##  Overview

NeuroWell is an AI-powered mental wellness assistant designed to help users track mental health, analyze trends, and receive personalized recommendations.

This system integrates:

*  Trend analysis
*  Goal tracking
*  AI-based recommendations

---

## Features

*  Mental health assessment system
*  Trend analysis dashboard
*  Goal tracking module
*  Recommendation engine
*  User authentication (Login/Register)
*  Fully responsive UI

---

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **AI Logic:** Rule-based / JS-based recommendation engine
* **Deployment:** GitHub Pages

---

##  Project Structure

```
NeuroWell-UI/
│── assets/           # Images & static files
│── components/       # Reusable UI components
│── css/              # Stylesheets
│── data/             # Dataset / JSON
│── js/               # Core logic & AI engine
│── frontend/         # UI pages
│── *.html            # Main pages
```

---

##  Installation & Setup

```bash
git clone https://github.com/YOUR_USERNAME/NeuroWell-UI.git
cd NeuroWell-UI
```

Run locally:

```bash
npx live-server
```

---

## 📊 Workflow

1. User logs in
2. Takes assessment
3. System analyzes responses
4. AI generates recommendations
5. Dashboard shows trends & goals
